# Sample Dataset Structure

Place one folder per person inside `dataset/`.

```text
dataset/
  Mustajab/
    Mustajab_001.jpg
    Mustajab_002.jpg
    Mustajab_003.jpg
  Aisha/
    Aisha_001.jpg
    Aisha_002.jpg
    Aisha_003.jpg
```

Use 20 to 50 clear images per person for better results. Keep one main face per training image.
